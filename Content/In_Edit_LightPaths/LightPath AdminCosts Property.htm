Set LightPath Admin Costs

The LightPath AdminCost property allows you to assign a "cost" value to each lightpath.  When you set the AdminCosts for the lightpaths in your design, you are in effect assigning a relative priority to each lightpath (higher cost = lower priority).  These priorities are factored into the decisions made by Modeler's routing algorithms.  The Set LightPath Admin Costs tool allows you to set the admin costs of a group of lightpaths by applying a simple equation.

To use this tool, select Edit, LightPaths, Admin Costs... from the menu.  The dialog shown below will appear.  

Set LightPath Admin Cost

To change lightpath admin cost values, follow the steps below:

  1. Select a Filter (optional) - If the list of lightpaths to select from is extensive, you have the option of applying one or more filters to pare down the list.  To select a filter, click to place a checkmark next to it then use the pull-down menu to select a value.
  2. Select one or more LightPaths - Click to select a lightpath from the filtered list or select multiple lightpaths by holding down the Ctrl or Shift key while you click.  To select the entire list, click on the Select All button.  To clear your selections, click on the Clear All button.
  3. Enter values into the Equation - Use the descriptions below to determine the desired values for each component of the admin cost equation.  Use the arrow keys or type in a value for each component and click OK to calculate admin costs for the selected lightpaths. 

Admin Cost Equation Components

 

Length Coefficient This value will be multiplied by lightpath length to calculate a length based cost. The default value is one as lightpath length must be a factor in the cost calculation.
Conduits Coefficient This value will be multiplied by the number of conduits a lightpath has to calculate a conduit based cost. The default value is zero which excludes conduits from the cost calculation.
Hops Coefficient This value will be multiplied by the number of fiber hops a lightpath has to calculate a fiber based cost. The default value is zero which excludes fiber hops from the cost calculation.
Offset This value for base cost will be added to values for length based cost, conduit based cost, and fiber based cost to calculate the AdminCost for each lightpath.  The default value is zero which excludes an offset from the cost calculation.
