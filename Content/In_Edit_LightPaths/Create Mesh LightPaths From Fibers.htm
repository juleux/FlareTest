Create Mesh LightPaths from Fibers

Use this tool to create lightpaths by drawing property information from the fibers present in your network.  To use this tool, select Edit, LightPaths, Create Mesh LightPaths From Fibers... from the menu.  The dialog shown below will appear.  

When the Skip any Fibers option is checked, a lightpath is created for each fiber in the network for which no pre-existing mesh lightpath exists (pre-existing ring lightpaths are ignored).  A pre-existing lightpath is any mesh lightpath that has the same start and end location as the fiber in question.  Select the Skip any Fibers option when you want to insure that at least one mesh lightpath exists for each pair of locations connected by fiber.  When the Skip any Fibers option is left unchecked, a mesh lightpath is created for each fiber in the network, regardless of the presence of pre-existing mesh lightpaths.  When ready, click OK to create lightpaths.

Create Mesh LightPaths from Fibers Dialog

Each newly created lightpath will appear in the object tree with a default name based on it's start and end locations and a set of default property values:

  • The StartLocation and EndLocation properties are set to those of the corresponding fiber.
  • The Length and ConduitID properties are set to those of the corresponding fiber. 
  • The AdminCost property is given a default value of 1.
  • The Type property is given the default LightPath Type value for the network.

You can use the Network Data Tool or the Property Editor to change these values as necessary.
