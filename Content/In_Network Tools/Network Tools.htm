Network Tools

The Modeler provides a variety of tools for the purpose of analyzing and editing network information.  These tools are independent of the algorithms and related tools the Modeler provides to aid with the design process. Using these tools you can:

  • Add objects to a network design
  • Compare the contents of two networks
  • Filter network data for viewing, editing, and exporting to Excel
  • View network statistics
  • Use editing tools to expand or sort demands
  • Use editing tools to create lightpaths and assign administration costs
  • Assign demands to Virtual Concatenation Groups
