Export a Network Data Report to Excel

Using the Network Data Tool, you can select a subset of objects within your network and export reports that contain information about the objects you selected.  The reports are stored in Excel format enabling you to edit and/or print them as desired.

To create a report, open the Network Data Tool and select the Reports tab.  Display the data you wish to export by following the steps outlined in the Display Network Data section of this help file.  When you click on the Save to Excel button, any data present in the table will be copied to an Excel file and saved.  Files are saved with filenames that reference the filename of the source network, as well as the date and time of export.  Files are stored in the Documents folder created during the installation process. 

Once the export process has been initiated, you can continue to work and export additional report data as desired.  When a report is being processed, any additional report requests are stored in a queue and processed in the order they were requested.  You can display and edit the queue by selecting Excel Export Queue Manager from the Tools menu.
