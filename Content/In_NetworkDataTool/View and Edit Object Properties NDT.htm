View and Edit Object Properties with the Network Data Tool

Within the Network Data Tool, you can display and edit the properties of an object of your choosing.  To do so, click on the Property Editor tab at the top of the Network Data Tool.  When you select an object from the list of filtered objects on the left side of the tool, it's properties will be displayed within the tab.  You can edit any property value that is displayed in black.  Properties shown in gray are status only.

You can also view and edit property values for multiple objects at the same time.  Start by selecting the desired objects (hold down the Cntl key and click on the desired objects or use Shift-Click to select a range of contiguous objects) from the Filtered Objects list and examining the property grid.  When multiple objects are selected and a property value is shown in black, then all of the selected objects share the same value for that property.  If multiple objects are selected and a property value is blank, then one or more of the selected objects has a different value for that property than the others do.   To edit the value of a property for multiple objects, select the objects and then replace the existing property grid entry (blank or black) with the desired value.
