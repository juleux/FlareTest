The Network Data Tool

The Modeler's Network Data Tool provides controls and filters that enable you to select a set of network objects (conduits, demands, fibers, lightpaths, locations, and rings are supported) and display a variety of information about them.  For example, you could select all of the lightpaths connected to a specific location and create a table of lightpath capacity information.  The table could be viewed and sorted within the dialog or exported to Excel for analysis and printing.  You can use this tool to:

  • Display data associated with one or more objects of your choosing. 
  • Apply object filters to pare down the list of objects that can be included in a data set.
  • Export a network data report to Excel.
  • View and edit an object's properties using the Property Editor.

To open the Network Data Tool, select Tools, Network Data Tool from the main menu or right-click on an object in the object tree and select Network Data Tool... from the pop-up menu.
