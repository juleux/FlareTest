Filter Network Data

Within the Network Data Tool, you can apply one or more filters to the list of objects available for selection.  To apply an object filter, click on the Filters tab at the top of the tool and follow the steps below:

  1. Select an Object Type - Use the pull-down menu to select an object type.  Filters that can be applied to the object type you chose will be available for selection within the Filters tab.  Filters that cannot be applied will be grayed out.
  2. Select one or more Filters - To select a filter, click to place a checkmark next to it then use the pull-down menu to select a value. You can make as many filter selections as are available.

As you make filter selections, the list of Filtered Objects will be updated to reflect your selections.  An object must satisfy all of the criteria you select (i.e. Type=OC48_STM16 AND Scope=Mesh AND StartLocation= PS-Node105) to appear in the Filtered Objects list.
