Excel Export Queue Manager

The Excel Export Queue Manager lists the reports currently slated for export to Excel from either the Network Data Tool or the Failure Analysis Results dialog.  To access the queue manager, select Tools, Excel Export Queue Manager...  from the main menu. 

The queue manager contains a list of each report and it's current status.  The report that is currently being processed is located at the bottom of the list.  The report next in line to be processed is located directly above it.  The report located at the top of the list will be processed last.  Each report is assigned a file name that it will be saved under.  Each file name contains the name of the Modeler file that the report data originated from, the title of the report, and the date and time the report was requested.

To cancel a report, click on the Cancel button located in the row in which the desired report is listed.
