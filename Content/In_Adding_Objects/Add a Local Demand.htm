Add a Local Demand

A local demand, also referred to as a local circuit, is a demand that starts and ends at the same location. 

To add a local demand to your design:

  1. Click on the Add Demand icon in the Object toolbar or select Edit, Add, Demand from the main menu.  The Add Demand cursor will appear.

  1. Define the new demand:  Within the network view, move the cursor over the desired locale until it's icon changes to a hand.  Click and release the mouse button once.  A new demand will be created and it's location will be highlighted in the network view (see below).

Add a Local Demand - Click and Release on the Start/End Location

  1. A new demand will appear in the network view (see below).

Add a Local Demand - Network View

  1. A new demand object will appear in the Object tree as a member of the Demands collection.  It's properties will be displayed in the Property editor. Each new local demand is given a default name based on the location it originates from and a set of default property values.  Using the Property editor, enter values for the demand's Quantity, Scope, and Type properties (see below).
