Add a Virtual Concatenation Group

To add a virtual concatenation group to your design:

  1. Click on the Add Virtual Concatenation Group icon in the Object toolbar or select Edit, Add, VirtualConcatenationGroup from the main menu.  The Add Virtual Concatenation Group cursor will appear.

  1. Define the new Virtual Concatenation Group: 

    With Distinct Endpoints - Within the network view, move the cursor over the first Endpoint of the virtual concatenation group until it's icon changes to a hand.  Click on the first Endpoint location (Baltimore), hold the mouse button down, and drag towards the second Endpoint location (NewLocation) of the virtual concatenation group.  A black line will appear as you reach the second Endpoint (see below).  When the black line shows the desired connection, release the mouse button.

    With Identical Endpoints - Within the network view, move the cursor over the Endpoint location of the virtual concatenation group until it's icon changes to a hand and click once.

Add a Virtual Concatenation Group - With Distinct Endpoints

  1. The endpoint(s) of the newly created virtual concatenation group will be highlighted in the network view (see below).

Add a Virtual Concatenation Group - Network View

A new virtual concatenation group object will appear in the Object tree as a member of the VirtualConcatenationGroups collection.  It's properties will be displayed in the Property editor. Each new virtual concatenation group is given a default name, based on its endpoint locations, and a set of default property values.  Using the Property editor, enter values for the virtual concatenation group's Structure property (see below).
