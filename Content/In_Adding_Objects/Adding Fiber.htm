Add a Fiber

To add a fiber to your design:

  1. Click on the Add Fiber icon in the Object toolbar or select Edit, Add, Fiber from the main menu.  The Add Fiber cursor will appear.

  2. In a network view, select a start location by clicking on the name of a locale (Baltimore). Your selection will be highlighted as shown below.

Add a Fiber - Select Start Location

  1. Click inside the selection box surrounding the start locale.  Hold the mouse button down and drag the towards the desired end locale (NewLocation).  A black line will appear as you drag (see below).  

Add a Fiber - Click and Drag from Start Location to End Location

  1. When the black line shows the desired connection, release the mouse button.  A fiber will appear in the network view (see below).

Add a Fiber - Network View

  1. A new fiber object will appear in the Object tree.  Its properties will be displayed in the Property editor. Each new fiber is given a default name, based on its start and end locations, and a set of default property values.   Using the Property editor, enter the desired values for the fiber's Name, Length, and AdminCost properties (see below).
