Add a Conduit

Conduits can be created in one of two ways:

  • Click on the Add Conduit icon in the Object toolbar or select Edit, Add, Conduit from the main menu.  A new conduit object will appear in the Object tree as a member of the Conduits collection.  The Modeler assigns a unique identification number to each new conduit object.
  • Assign a conduit to a lightpath or fiber by entering a number in the object's ConduitID property.  If the number you enter is not an existing ConduitID, the Modeler will create a conduit object and set it's Id property to your entry.  Use this method when you want to assign an id number of your own choosing.
