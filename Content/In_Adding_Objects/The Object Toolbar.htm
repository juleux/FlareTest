The Object Toolbar

The Object toolbar is located under the Main toolbar at the top of the Modeler's main window.  It contains icons for the tools you need to add network objects (i.e. locations, lightpaths, demands) to your design.   To view a tool's description, hover your cursor over it's icon in the toolbar for a few seconds and a tool tip will appear.
