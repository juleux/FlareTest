Add a Location

When you add a location to your design, the Modeler automatically creates a corresponding locale object for each network view.

To add a location to your design:

  1. Click on the Add Location icon in the Object toolbar or select Edit, Add, Location from the main menu.  Position the Add Location cursor over the network view and click on the desired coordinates.  The dialog shown below will appear:

  1. Enter a name and click OK.  A locale object will appear at the coordinates you selected (see below).  Locale objects are assigned an icon based on the type of location (i.e. SN16000) they represent.  If you frequently add new locations, you may want to select a location type as the default type for every new location that is created.  To select a default location type, click on the network filename in the Object tree.  Within the Property editor, use the pull-down to select a value for the location type property.  To learn more about setting defaults, read the Set Default Object Properties section of this help file.

Add a Location - Network View

  1. The Object tree will be updated to include a new location object and a locale object of the same name for each network view (see below).  Using the Property editor, edit the location's Type property as required.
