Set Default Object Properties

When an object is added to a network, the Modeler sets one or more of it's property values (depending on the type of object) by entering default values provided by you.  Examples of properties you can set default values for include: DemandQuantity, LightPathType, LocationType, and RingArchitecture.  To enter default property values, click on the network filename in the Object tree.  Next, use the Property editor to enter the desired defaults as shown below.
