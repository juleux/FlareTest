Add an Ethernet Service

Ethernet Service objects function as children of Virtual Concatenation Group objects.  You can add an Ethernet Service object to any network that contains at least one VCG.  To add an ethernet service object to your design:

  1. Click on the Add Ethernet Service icon in the Object toolbar or select Edit, Add, EthernetService from the main menu.  The Add New EthernetService dialog will appear.

  1. Use the top pulldown to select the Virtual Concatenation Group you want the Ethernet Service assigned to.  The pulldown contains a list of the VCGs present in the network.  Each time a VCG selection is made, the VirtualConcatenationGroupEndPoint and EthernetServiceName fields are updated to provide relevant pulldown selections.

Add New Ethernet Service Dialog

  1. Select the EndPoint you want the Ethernet Service assigned to.  The pulldown contains the endpoints of the currently selected VCG.
  2. Enter a name for the new Ethernet Service or use the default name provided.  The default name contains the prefix ES_ followed by the name of the currently selected VirtualConcatenationGroup and EndPoint.  Click OK to add the Ethernet Service.
  3. A new ethernet object will appear in the Object tree as a child of the virtual concatenation group you selected.  It's properties will be displayed in the Property editor.
