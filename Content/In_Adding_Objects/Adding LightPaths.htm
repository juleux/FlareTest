Add a LightPath

There are two methods for adding a lightpath to your design.  The Fiber Selection method and the Location Only method.  Use the Fiber Selection method when you want to select the individual fibers over which a lightpath will travel.  Use the Location Only method to create a lightpath along with a conduit for it to travel over.  Lightpaths added using this method are created without fiber assignments.

Fiber Selection Method

To use the Fiber Selection method to create a lightpath, you need the following information:

  • The lightpath's start and end locations
  • The sequence of fibers you want the lightpath to travel over

When you use this method to add a lightpath to your design, the locations and fibers you select cannot be edited directly.  Instead, you must delete the lightpath in question and create a new lightpath with the correct information. 

To add a lightpath to your design by selecting fibers:

  1. Click on the Add LightPath icon in the Object toolbar or select Edit, Add, LightPath from the main menu.  The Add LightPath cursor will appear.

  2. Select the desired sequence of fibers:  Press and hold the Shift key.  While continuing to hold the Shift key, click on each fiber in order (Fiber 1 then Fiber 2, etc.) from the Start locale (Baltimore) to the End locale (Charlotte) (see below).

 

Note:  In some situations (i.e. multiple fibers and lightpaths in a concentrated area), it may be difficult to pinpoint a specific fiber in the network view.  In these instances it is helpful to hide the lightpath layer while performing this procedure.   To do so, set the LightPathRouting property of the current view to None.

Add a LightPath - Select Fibers from Start to End

  1. Define the new lightpath:  Move the cursor over the Start locale of the lightpath until it's icon changes to a hand.  Click on the Start locale (Baltimore) of the lightpath.  Hold the mouse button down and drag towards the End locale (Charlotte) of the lightpath.  A black line will appear as you drag.  When the black line shows the desired connection, release the mouse button (see below). 

Add a LightPath - Define LightPath
  1. A new lightpath will appear in the network view (see below).  If a lightpath does not appear, verify that the fibers were selected in the proper order and that the path you chose was valid.  

 

Note:  If you turned the lightpath layer off to perform this procedure, the lightpath will not appear until you set the current view's LightPathRouting property to Direct.

Add a LightPath - Network View

  1. A new lightpath object will appear in the Object tree.  It's properties will be displayed in the Property editor. Modeler gives each new lightpath a default name, based on it's start and end locations, and a set of default property values.  The LightPath Length property defaults to the sum of the length of the lightpath's fibers.  Using the Property editor, enter values for the lightpath's Length, AdminCost, and Type properties (see below).

Location Only Method

To use the Location Only method to create a lightpath, the only information you need is the lightpath's start and end locations.  When you use this method to add a lightpath to your design, the following is true:

  • Fiber selections cannot be added in the future.
  • The start and end locations you selected cannot be edited directly.  Instead, you must delete the lightpath in question and create a new lightpath with the correct information. 

To add a lightpath to your design without selecting fibers:

  1. Click on the Add LightPath icon in the Object toolbar or select Edit, Add, LightPath from the main menu.  The Add LightPath cursor will appear.

  2. Define the new lightpath: Within the network view, move the cursor over the Start locale of the lightpath until it's icon changes to a hand.  Click on the Start locale (Baltimore), hold the mouse button down, and drag towards the End locale (NewLocation) of the lightpath.  A black line will appear as you drag.  When the black line shows the desired connection, release the mouse button.

Add a LightPath without Selecting Fibers - Click and Drag from Start Location to End Location

  1. A new lightpath will appear in the network view (see below).  A new conduit object will be created and assigned to the lightpath.

Add a LightPath without Selecting Fibers - Network View

  1. A new lightpath object will appear in the Object tree.  It's properties will be displayed in the Property editor. The Modeler gives each new lightpath a default name, based on it's start and end locations, and a set of default property values.  The ConduitId property is set to the Id of the newly created conduit.  Using the Property editor, enter values for the lightpath's Length, AdminCost, and Type properties.
