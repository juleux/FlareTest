Add a Demand

Before adding a demand, verify that its start and end locations are connected by fiber or lightpaths before executing the steps that follow.  To view demands, set the DemandRouting property of your network view to Direct. 

To add a demand to your design:

  1. Click on the Add Demand icon in the Object toolbar or select Edit, Add, Demand from the main menu.  The Add Demand cursor will appear.

  1. Define the new demand:  Within the network view, move the cursor over the Start locale of the demand until it's icon changes to a hand.  Click on the Start locale (Baltimore), hold the mouse button down, and drag towards the End locale (NewLocation) of the demand.  A black line will appear as you drag (see below).  When the black line shows the desired connection, release the mouse button.

Note:  If a black line does not appear, you may have inadvertently created a local demand.  If so, it will be highlighted within the object tree and have a default name in the format D_x_x, where x is the name of your intended start location.  If this is the case, simply delete the demand you created and repeat this step making sure that you hold the mouse button down after clicking on a locale.

Add a Demand - Click and Drag from Start Location to End Location

  1. A new demand will appear in the network view (see below).

Add a Demand - Network View

  1. A new demand object will appear in the Object tree as a member of the Demands collection.  It's properties will be displayed in the Property editor. Each new demand is given a default name, based on its start and end locations, and a set of default property values.  Using the Property editor, enter values for the demand's Protection Type, Quantity, Scope, and Type properties (see below).
