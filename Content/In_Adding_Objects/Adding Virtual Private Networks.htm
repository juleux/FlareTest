Add a Virtual Private Network

Virtual Private Networks (VPNs) can be created in one of two ways:

  • Click on the Add VPN icon in the Object toolbar or select Edit, Add, VPN from the main menu.  A new VPN object will appear in the Object tree as a member of the VPNs collection.  The Modeler assigns a unique identification number to each new VPN object.
  • Assign a VPN to a lightpath or demand by entering a number in the object's VPN property.  If the number you enter is not an existing VPN, the Modeler will create a VPN object and set it's Id property to your entry.  Use this method when you want to assign an id number of your own choosing.
