Install a License

To install a new license, start the Modeler then select File, License, Install from the main menu.  A file selection dialog (see below) will appear.  Use the dialog to browse for and select the desired license file (.lic extension) and click the Open button to install it.  

License Installation Dialog

When your license is within two weeks of expiration, the Modeler displays a warning message upon start-up to indicate that an updated license should be obtained.  For instructions on obtaining a new license, refer to the Obtain a New License section of this help file.

Note:  A basic license is provided with each copy of the Modeler that is purchased.  With a basic license, you can use all but the most advanced features of the Modeler.  Advanced features include support for ring topologies, failure analysis algorithms, and network data import from .  If you have a need for any of these features, contact your representative to request an enhanced license.
