Install the Modeler

Prior to installing the Modeler, check to see that your system meets the minimum hardware and software requirements.  The procedure for installing the Modeler on Windows 2000 and XP systems consists of the following steps:

  1. Download operating system update files from Microsoft's website
  2. Install operating system update files on your computer system
  3. If present, remove an existing version of the Modeler from your computer
  4. Run the Modeler Setup wizard

When you install the Modeler for the first time, it's functionality is limited to accessing the help file and accessing the menu items necessary to obtain and install a license.  To use the full functionality of the Modeler, you must install a machine specific license provided by . For instructions on obtaining and installing a license file, refer to the Obtain a New License and Install a License sections of this help file.

Download Operating System Update Files

To begin the installation process, you must download two update files from Microsoft's Download Center website.  Additional downloads may be required if your operating system has not been recently updated.  The required downloads include:

Windows Installer 3.1 Redistributable (v2)
  • This update is not needed if your computer has Windows XP, Service Pack 2.

Microsoft .NET Framework Version 2.0 Redistributable Package (x86)

Click on a filename above to link to it's download page on the Microsoft website.  Once there, read the system requirements section to determine if additional updates (i.e. service packs) are necessary.  Otherwise, proceed with the download.

After you download each file to your hard disk, continue by following the installation instructions below.

Install Operating System Update Files

Once the update files are downloaded, perform an update by completing the following steps, in order:

  1. Install Windows Installer update, as needed, by double-clicking on the download file.
  2. Install .NET Framework by double-clicking on the download file.

Once completed, this portion of the installation process does not have to be repeated for subsequent installs, unless new versions of these files are available.

Remove An Existing Version of the Modeler

If an earlier version of the Modeler is present on your computer, it must be removed before a newer version can be installed.  To remove Modeler, perform the following steps:

  1. From the Start menu of your computer, select Settings, Control Panel, Add/Remove Programs

  2. Click to select Modeler from the list of programs

  3. Click on Remove

Run the Modeler Setup Wizard

The setup wizard is a standard Windows install.  To run the wizard, double-click on Modeler (version number). Setup.msi.  A dialog will appear to guide you through the installation process.  The dialog will prompt you to choose the directory where the Modeler will be installed. The default directory is \Program Files\\Modeler.  For a description of each sub-directory created during the install procedure, read the Directory Structure section of this help file.

To start the Modeler, click on the Start menu, followed by Programs, , and then Modeler.
