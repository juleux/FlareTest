Start the Modeler

To run the Modeler from the Start menu, click on Programs, then Modeler.  You can also run the program by double-clicking on Sycamore.Modeler.GUI.exe in the install directory.  To help you get started, sample input files (PFL Mesh.xml, PFL Ring.xml) are provided in the Documents folder located in the install directory.
