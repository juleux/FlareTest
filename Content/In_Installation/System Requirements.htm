System Requirements

Your computer system must satisfy the following requirements in order for you to install and run the Modeler. 

Operating System Microsoft Windows 2000 or XP
Memory 256 MB minimum, 512 MB recommended
CPU 1 GHz or greater is recommended
Hard Disk Space 500 MB minimum, 1 GB recommended
Screen Resolution 1024x768 or greater is recommended
Application Software Internet Explorer 5.01 or later
Microsoft Excel ®  for reporting
