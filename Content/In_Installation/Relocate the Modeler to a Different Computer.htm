Relocate the Modeler to a Different Computer

For each copy of the Modeler, provides a license that enables you to use it on a specific computer.  As a result, you must obtain a new license when you want to use the Modeler on a different computer.  To move your Modeler application to a new computer, follow the procedure below:

  1. On the existing system, start the Modeler then select File, License, Uninstall from the main menu.
  2. The License Status dialog, containing relevant system information, will be displayed.
  3. Within the License Status dialog, select File, Save to store your system information to a file.  You may want to include the word "source" in the filename.
  4. Install the Modeler application on the destination computer.
  5. On the destination computer, start the Modeler then select File, License, Status from the main menu.
  6. Within the License Status dialog, select File, Save to store your system information to a file.  You may want to include the word "destination" in your filename.
  7. Attach both files to an email request for a new license.  Address your email to at designtools@sycamorenet.com.
  8. When you receive a new license from , follow the instructions in the Install a License section of this help file.

 

Note:  Use the uninstall command only when needed to change computer systems.  Once a license has been uninstalled, it is no longer valid and cannot be reinstalled.  Therefore, we recommend that you avoid uninstalling unless you have immediate plans to obtain a new license.
