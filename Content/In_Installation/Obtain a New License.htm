Obtain a New License

Each Modeler installation requires a machine specific license to operate.  Upon installation, each license is activated to provide functionality for a pre-determined period of time.  Each time you start the Modeler, it evaluates the current license and displays a message if the license is no longer valid.  To obtain a new license from , follow the procedure below:

  1. Start the Modeler then select File, License, Status from the main menu to open the License Status dialog.  The dialog (see example below) contains the information needed to generate a new license.
  2. Within the Status dialog, select File, Save and enter a filename.
  3. Attach this file to an email request for a new license.  Address your email to at or contact your local representative.

When you receive a new license from Sycamore, follow the instructions in the Install a License section of this help file.
