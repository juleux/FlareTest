Directory Structure

During the installation process, you were prompted to choose a destination directory for the Modeler.  Within the directory you chose, several sub-directories were created for use by the Modeler.  A description of each sub-directory is provided in the table below.  These sub-directories are required for the Modeler to work properly and you should not rename or delete any of them.  You may uninstall the Modeler at any time and all of the files and folders written to your system during installation will be removed.  Any new files created by you will remain behind, possibly causing the parent directories to remain.

Cursors Custom cursor files are stored in this location.  These cursors may be modified if desired.
Documents Default location to store network files.  Network files can be stored anywhere.
NetworkViews/Backgrounds Files that contain background images used in NetworkViews are stored in this directory.  These files may be modified.
NetworkViews/Icons Files that contain location icon images used in NetworkViews are stored in this directory.  These files may be modified.
