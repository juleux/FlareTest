Apply Property Filters When Comparing Networks

The Network Compare tool provides multiple sets of filters, a set for each object type (i.e. demand, location, lightpaths), a set for all objects combined, and a set for all linked objects.  Each filter in a set corresponds to an object property.  When you click to apply a filter, the Modeler will conduct a comparison that ignores the property values associated with the filter.  The affect of applying a location property filter is illustrated in the before and after screen captures provided below. 

To apply a property filter, select a set of filters by clicking on the desired tab in the Object Property Filter control.  Next, place a checkmark next to the property you want to exclude from (filter out of) the comparison.  Multiple filters may be used at the same time.  The Modeler updates the comparison results whenever the status of a filter is changed.
