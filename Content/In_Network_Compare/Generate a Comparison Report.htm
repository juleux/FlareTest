Generate a Network Comparison Report

Using the Network Compare tool, you can select the criteria by which two networks (referred to as Network A and Network B) will be compared and create a report containing the comparison results.  Each report contains the following information:

  • Common Mismatches List - contains the names of objects that appear in both networks but have differences in one or more property values.  The values of the properties that differ are included.
  • Common Matches List - contains the names of objects that appear in both networks and share the same property values.
  • Objects Unique to Network A - contains the names of objects that appear in network A but not in network B.
  • Objects Unique to Network B - contains the names of objects that appear in network B but not in network A.

To generate a network comparison report:

  1. Open the first network you want to use in the comparison (Network A).
  2. Open the Network Compare Tool.
  3. Select the second network you want to use in the comparison (Network B).
  4. Apply the desired object and property filters.
  5. Click on the Comparison Report button at the bottom of the Network Compare dialog.

When these steps are completed, the Modeler will create a text file containing the same information present in the Network Compare dialog.
