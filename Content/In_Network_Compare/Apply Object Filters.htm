Apply Object Filters When Comparing Networks

The Network Compare tool provides a filter for various types of objects (i.e. locations, lightpaths, failure scenarios).  When you click to apply an object filter, the Modeler will ignore all objects of the type you selected when conducting the comparison. The affect of applying a Locations object filter is illustrated in the before and after screen captures provided below. 

To apply an object filter, locate the Object Filter control at the bottom of the dialog then place a checkmark next to the type of object you wish to exclude from (filter out of) the comparison.  Multiple filters may be used at the same time.  The Modeler updates the comparison results whenever the status of a filter is changed.
