Compare Networks

The Modeler's Network Compare tool provides controls and filters that can be used to compare the currently open network with a second network of your choosing.  Using this tool you can:

  • Display network comparison results which include: a list of objects unique to each network and a list of objects common to both networks.  The list of objects being compared includes: locations, fibers, rings, lightpaths, demands, network statistics, and failure scenarios.
  • Apply object filters to limit comparison results to the objects you want to focus on.
  • Apply property filters to limit comparison results to the object properties you want to focus on.
  • View properties for objects in either network.  Edit property values for objects in the current network (Network A).
  • Generate a Network Comparison Report to document comparison results.

For instructions on how to get started with the Network Compare tool, read the Display Network Comparison Results section of this help file.
