View and Edit Object Properties with the Network Compare Tool

Within the Network Compare tool, you can view the properties of objects contained within the networks (referred to as Network A and Network B) that you are comparing.  In addition, you can edit the properties of objects contained within the current network (Network A).

Instructions fro viewing and editing properties are as follows:

To view or edit the properties of an object in Network A: Click on an object in the Objects Unique to A list or the Common Objects list.  If necessary, click on A in the Select Network control beneath the property grid.  The properties of the object you selected will be listed in the property grid.  You can edit any property value that is displayed in black.  Properties shown in gray are status only.

To view the properties of an object in Network B: Click on an object in the Objects Unique to B list or the Common Objects list.  If necessary, click on B in the Select Network control beneath the property grid.  The properties of the object you selected will be listed in the property grid.  

To view or edit the properties of multiple objects: Start by selecting the desired objects from one of the three available lists (hold down the Cntl key and click on the desired objects or use Shift-Click to select a range of contiguous objects) and examine the property grid.  When multiple objects are selected and a property value is shown in black, then all of the selected objects share the same value for that property.  If multiple objects are selected and a property value is blank, then one or more of the selected objects has a different value for that property than the others do.  To edit the value of a property for multiple objects in Network A, select the objects and then replace the existing property grid entry (blank or black) with the desired value.

Note: While the dialog does not prevent you from editing property values in Network B, any changes you make are lost when the dialog is closed.

Note: Using this dialog to change the Name property of objects is not recommended.  Instead, close the dialog and make your edits in the property editor located at the bottom of the Main window.
