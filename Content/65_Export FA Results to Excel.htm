Export Failure Analysis Results to Excel

Prior to creating a Failure Analysis Results report, you must first generate results data by using the Failure Analysis Tool to perform either a manual or an automatic failure analysis.  When the analysis is complete, click on the Results tab of the Failure Analysis Tool to display the results data tables. 

You have two options when generating a Failure Analysis Results report:

  • Export All of the Results Tables to an Excel File - To create a comprehensive Failure Analysis Results report, place a checkmark next to the Save All Tables option at the top of the Results tab and click the Save to Excel button.  Each table of results data will be copied to a separate worksheet within a single Excel file. 

Note that this can be a time consuming report to create when large amounts of results data are present.

  • Export the Selected Results Table to an Excel File - Within the results tab, display the type of results data you wish to export: LightPaths, Failure Scenarios, Affected LightPaths, Iteration Details, Affected Demands, Paths, or Affected Virtual Concatenation Groups.  If necessary, click to deselect the Save All Tables option at the top of the Results tab.  When you click on the Save to Excel button, an Excel worksheet will be created and saved.  The worksheet will contain data of the same type on display.  

Note that while the results data table may contain a subset of data for a particular failure scenario, the results data for every failure scenario will be exported.  For example, when viewing Affected Demands, the results table contains the demands for a single failure scenario, however, the exported file contains the Affected Demands for every failure scenario that was analyzed.

Failure Analysis Results - Export to Excel

Failure Analysis reports are saved with filenames that reference the filename of the source network, the type of data in the report, and the date and time of export.  Files are stored in the Documents folder created during the installation process. 

Once the export process has been initiated, you can continue to work and export additional report data as desired.  When a report is being processed, any additional report requests are stored in a queue and processed in the order they were requested.  You can display and edit the queue by selecting Excel Export Queue Manager from the Tools menu.
