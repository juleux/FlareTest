Network Statistics Report

This report is provided as a worksheet entitled "Network Statistics".  It contains the information described in the table below.

Type of Data Sub-Categories Property Name(s) Description
All Types Demands, LightPaths, Rings, Fibers, Locations, Ethernet Services, VCGs, Conduits, VPNs Count The number of objects of each sub-category that are present in the network.
  Demands, LightPaths, Rings, Fibers, Locations Locked, Unlocked The number of objects of each sub-category that are locked/unlocked in the network.  Locking an object prevents it from being deleted until it is unlocked.
Demand Total, OnePlusOne, SharedPath, Unprotected, Ring Count The number of demands of each sub-category that are present in the network.
    UnRouted/Routed The number of routed/un-routed demands in the network.
    Fixed Routing The number of demands that have fixed routing in the network.
    Node Diverse, Conduit Diverse, Diverse The number of routed demands that are node/conduit diverse.
    Equivalent STS1 The total number of demands in STS1 units.
LightPath Traffic   Units The units used for the lightpath traffic values below.
    Working, Protect Dedicated, Protect Shared The amount of each type of traffic routed over lightpaths in the network.
    Total Used The total amount of traffic routed over lightpaths in the network.
    Available The amount of traffic available for routing.
    Total The total amount of traffic, used and available, within the network.
    Utilization The percentage of lightpath capacity in use.
LightPath Capacity Total, Mesh, Ring Maximum Type The largest type of lightpath present in the network.  Used to determine the units for the capacity related values below.
    Capacity Sum of working and protect path traffic for the lightpaths in the network.  Units are based on the Maximum Type entry above.
    Capacity Distance Sum of capacity distance for all lightpaths in the network.  Capacity distance is the capacity of a lightpath multiplied by it's length.
    Capacity Incremental The difference between Capacity and CapacityMinimum.
    Capacity Over Maximum The amount by which maximum capacity has been exceeded.
Rings   Stack Count The quantity of stacked rings.
Fibers   Total Length The sum of the lengths for all fibers in the network.
    Used Length The sum of the lengths of all fibers in use in the network.
Locations   MaxFiberDegree The maximum fiber degree within the network, where fiber degree is the number of fibers that terminate at a given location.
    MaxLightPathDegree The maximum lightpath degree within the network, where lightpath degree is the number of lightpaths that terminate at a given location.
    MaxRingDegree The maximum ring degree within the network, where ring degree is the number of rings that terminate at a given location.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
