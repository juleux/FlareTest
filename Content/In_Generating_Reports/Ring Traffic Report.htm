Ring Traffic Report

This report is provided as a worksheet entitled "Ring Traffic".  It contains the information described in the table below.  Use this report to evaluate traffic flow for one or more rings in your proposed design.

Type of Data Column Name(s) Description
Ring Name, LongName, Locked This is user supplied information that uniquely describes each ring.
   Architecture This is user supplied information for ring architecture.  Valid entries include:
  • BLSR2F_MSSSPRING2F
  • BLSR4F_MSSSPRING4F
  • ASYM4F
  • UPSR_SNCP
   Type This is user supplied information for ring type.  Valid entries include:
  • OC3_STM1
  • OC12_STM4
  • OC48_STM16
  • OC192_STM64
   StackCount The quantity of stacked rings.
   MaximumCapacity The maximum capacity of all lightpaths associated with a ring.
Traffic Working Maximum The maximum working capacity of a ring based on the maximum working traffic over all ring lightpaths.  
   Capacity The amount of working capacity based on the maximum working traffic over all ring lightpaths.  
   GroomingFactor A function of ring type and architecture which is used to calculate the size of a ring.
   Circumference The total length of the lightpaths that comprise a ring.
   NumberOfNodes The number of locations that add or drop traffic within a ring.
Location Locations A list of the locations (comma delimited) that comprise a ring.
Fiber Fibers A list of the fibers (comma delimited) that comprise a ring.
LightPath LightPaths A list of the lightpaths (comma delimited) that comprise a ring.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
