Locations LightPath Pairs Traffic Report

This report is provided as a worksheet entitled "Locations LightPath Pairs Traff".  It contains the information described in the table below.  Use this report to evaluate traffic flow for lighpath pairs in your proposed design.

Type of Data Column Name(s) Description
  Location This is the name of a location. 
LightPaths LightPath1, LightPath2 The names of the first and second lightpath in a pair that are linked to a given location.
Traffic Working, ProtectDedicated, ProtectShared, Available, Total The amount of each type of traffic routed over the lightpath pair. Units (dependent on network type) are displayed at the top of each column of traffic data.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
