Location LightPath Degree Report

This report is provided as a worksheet entitled "Location LightPath Degree".  It contains the information described in the table below.  

Note:  An Excel worksheet can contain a maximum of 256 columns.  This report will not be generated if more than 256 columns are required based on the use of lightpaths in the network.

 

Type of Data Column Name(s) Description
Location Name, Hub This is user supplied information that uniquely describes each location.
All LightPath Degrees LightPath Degree The number of lightpaths associated with a location.
Signal Type (for all lightpath degrees) OC-1, OC-3, OC-12, OC-48, OC-192 The number of signals of each type that are dropped at (passed through) a location. 
LightPath Degree 1, 2 ... LightPath Name The name of each lightpath associated with a location.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
