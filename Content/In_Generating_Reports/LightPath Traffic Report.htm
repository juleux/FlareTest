LightPath Traffic Report

This report is provided as a worksheet entitled "LightPath Traffic".  It contains the information described in the table below.  Use this report to evaluate traffic flow for one or more lightpaths in your proposed design.

Type of Data Column Name(s) Description
LightPath Name, LongName, Locked, StartLocation, EndLocation, Cost, Length, ConduitIDs This is user supplied information that uniquely describes each lightpath.
   Type This is user supplied information for lightpath type.  Valid entries include:
  • OC3_STM1
  • OC12_STM4
  • OC48_STM16
  • OC192_STM64
   Domain The ring this lightpath is associated with.
   VPN The virtual private network (VPN) this lightpath is assigned to.
   TrunkIDs The list of trunk identification numbers associated with this lightpath.  TrunkID information is imported from . Individual ids are separated by colons.
Capacity CapacityMinimum, CapacityMaximum The minimum/maximum capacity of a lightpath.
   Capacity Sum of working and protect path traffic for a lightpath.  Units are based on Type entry above.
   CapacityIncremental The difference between Capacity and CapacityMinimum.
   CapacityMaximumExceeded True if maximum Capacity has been exceeded.
   Utilization The percentage of lightpath capacity in use.
   TimeSlotsStatus The status of each timeslot in the lightpath.  Status types include Available, Working, ProtectDedicated, and ProtectShared.
Traffic Vectors Working, ProtectDedicated, ProtectShared, Available The quantity of time slots of each signal type assigned to the demand's working (protect dedicated, etc.) path in the format of Type1:Type3:Type12:Type48:Type192 where
  • Type1 = OC1_STS1_VC3_DS3_E3_AU3
  • Type3 = OC3_STS3_STM1_VC4
  • Type12 = OC12_STS12_STM4
  • Type48 = OC48_STS48_STM16
  • Type192 = OC192_STS192_STM64
Fiber Fibers The sequential list of fibers (comma delimited) over which a given lightpath is routed.
   Fiber Count The number of fibers over which a given lightpath is routed.
Traffic Working, ProtectDedicated, ProtectShared, Available, Total The amount of each type of traffic routed over a lightpath. Units (dependent on network type) are displayed at the top of each column of traffic data.
Traffic Distance Working, ProtectDedicated, ProtectShared, Available, Total The distance over which each type of traffic travels.  Units (dependent on network type) are the same as other traffic data.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
