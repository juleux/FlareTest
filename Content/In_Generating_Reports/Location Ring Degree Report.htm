Location Ring Degree Report

This report is provided as a worksheet entitled "Location Ring Degree".  It contains the information described in the table below.  

Note:  An Excel worksheet can contain a maximum of 256 columns.  This report will not be generated if more than 256 columns are required based on the use of rings in the network.

 

Type of Data Column Name(s) Description
Location Name, Hub This is user supplied information that uniquely describes each location.
All Ring Degrees Ring Degree The number of rings associated with a location.
    Stack Count The quantity of stacked rings.
    Dropped Traffic The total quantity of traffic that is dropped at a location. Units (dependent on network type) are displayed at the top of each column of traffic data.
    Through Traffic The total quantity of traffic that passes through a location. Units (dependent on network type) are displayed at the top of each column of traffic data.
     Ring Interconnect Traffic The total quantity of traffic that passes between adjacent rings. Units (dependent on network type) are displayed at the top of each column of traffic data.
Ring Degree 1, 2 ... Ring Name The name of each ring associated with a location.
    Ring Architecture The architecture of each ring associated with a location.  Valid entries include:
  • BLSR2F_MSSPRING2F
  • BLSR4F_MSSPRING4F
  • ASYM4F
  • UPSR_SNCP
    Ring Type The type of each ring associated with a location.  Valid entries include:
  • OC3_STM1
  • OC12_STM4
  • OC48_STM16
  • OC192_STM64
    Stack Count The quantity of stacked rings.
    Dropped Traffic The number of signals within the specified ring that are dropped at a location. Units (dependent on network type) are displayed at the top of each column of traffic data.
    Through Traffic The number of signals within the specified ring that pass through a location. Units (dependent on network type) are displayed at the top of each column of traffic data.
     Ring Interconnect Traffic The total traffic that passes between adjacent rings. Units (dependent on network type) are displayed at the top of each column of traffic data.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
