Generate Reports

The Modeler provides a number of reports to enable you to document your network design.  Reports are generated using Microsoft Excel® which must be properly installed on your computer system.  The Modeler provides two tools for generating reports, the Network Data Tool and the Reporting Tool.  Use the Reporting Tool when you want to create reports that incorporate all of the objects in your network of a particular type (i.e. lightpaths, locations).  Use the Network Data Tool when you want to create a report for a subset of individual objects selected by you.  This help section provides instructions for using the Reporting Tool and describes the data contained in each report.  For instructions on using the Network Data Tool, refer to the Export Network Data to Excel section of this help file.

Using the Reporting Tool, you can print the following types of reports:

  • Conduit Traffic
  • Location Traffic
  • Fiber Traffic
  • LightPath Traffic
  • Ring Traffic
  • Locations LightPath Pairs Traffic
  • Location Fiber Degree
  • Location LightPath Degree
  • Location Ring Degree
  • Demand Routing
  • Network Statistics
  • Trunks
  • APS MSP Protection Groups

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu.  The dialog shown below will appear:     

Click the checkbox next to the report(s) you wish to generate then click OK.  The Modeler will create each report as a single Excel worksheet and prompt you for a filename to save to.  

Note: Reports are only available for selection when the type of object they reference (i.e. ring, fiber) is present in the network design.  Otherwise, they are grayed out.
