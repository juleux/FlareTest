Trunks Report

This report is provided as a worksheet entitled "Trunks".  It contains the information described in the table below.  Use this report when you want to view network trunk data imported from .

Type of Data Column Name(s) Description
LightPath Name, Description This is user supplied information that uniquely describes each lightpath.
EndPoint 1,2 RouterID The id number of the router located at each of the trunk's two endpoints.
  Chassis The number of the chassis located at each of the trunk's two endpoints.
    Slot The number of the slot located at each of the trunk's two endpoints.
  Port The number of the port located at each of the trunk's two endpoints.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
