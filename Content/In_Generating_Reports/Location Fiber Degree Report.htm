Location Fiber Degree Report

This report is provided as a worksheet entitled "Location Fiber Degree".  It contains the information described in the table below.  

Note:  An Excel worksheet can contain a maximum of 256 columns.  This report will not be generated if more than 256 columns are required based on the use of fibers in the network.

 

Type of Data Column Name(s) Description
Location Name, Hub This is user supplied information that uniquely describes each location.
All Fiber Degrees Fiber Degree The number of fibers associated with a location.
Signal Type (for all fibers degrees) OC-3, OC-12, OC-48, OC-192 The number of signals of each type that are dropped at (passed through) a location. 
Fiber Degree 1, 2 ... Fiber Name The name of each fiber associated with a location.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
