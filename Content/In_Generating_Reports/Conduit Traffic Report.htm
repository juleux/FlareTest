Conduit Traffic Report

This report is provided as a worksheet entitled "Conduit Traffic".  It contains the information described in the table below.  Use this report to evaluate traffic flow for one or more conduits in your proposed design.

Type of Data Column Name(s) Description
Conduit ID The unique identification number for each conduit.
  Description A user supplied description of this conduit.
   Capacity The sum of the capacity values for the lightpaths assigned to this conduit.
   CapacityIncremental The sum of the incremental capacity values for the lightpaths assigned to this conduit.
Traffic Working, ProtectDedicated, ProtectShared, Available, Total The amount of each type of traffic routed over the lightpaths assigned to this conduit.  Units (dependent on network type) are displayed at the top of each column of traffic data. 
   Utilization The percentage of capacity in use for lightpaths assigned to this conduit. 
Hosted Items Fibers, LightPaths, Rings The sequential list of fibers, lightpaths, or rings (comma delimited) assigned to this conduit.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
