Fiber Traffic Report

This report is provided as a worksheet entitled "Fiber Traffic".  It contains the information described in the table below.  Use this report to evaluate traffic flow for one or more fibers in your proposed design.

Type of Data Column Name(s) Description
Fiber Name, LongName, Locked, StartLocation, EndLocation, Cost, Length, ConduitIDs This is user supplied information that uniquely describes each fiber.
Traffic Working, ProtectDedicated, ProtectShared, Available, Total The amount of each type of traffic routed over a fiber. Units (dependent on network type) are displayed at the top of each column of traffic data.
   Utilization The percentage of capacity in use for lightpaths assigned to this fiber. 
LightPaths OC-3, OC-12, OC-48, OC-192 The amount of each type of LightPath traffic hosted by this fiber.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
