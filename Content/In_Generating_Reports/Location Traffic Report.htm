Location Traffic Report

This report is provided as a worksheet entitled "Location Traffic".  It contains the information described in the table below.  Use this report to evaluate traffic flow for one or more locations in your proposed design.

Type of Data Column Name(s) Description
Location Name, LongName, Locked, Longitude, Latitude, CLLI, Building, Hub, Type This is user supplied information that uniquely describes each location.
   SwitchIPAddress, RouterID This information, imported from , identifies the ip address and router identification number of a location.
   PermitThroughTraffic This is user supplied information that tells the routing algorithms the types of lightpaths that will accept through traffic (traffic that does not terminate at a location). Valid entries include:
  • AnyLightPath
  • ExpressLightPaths
  • None
Degrees Fiber, LightPath, Ring The total number of fibers, lightpaths, or rings (of all types) associated with a location.
Dropped Traffic RingWorking, MeshWorking, MeshProtectDedicated, MeshProtectShared, Total The amount of each type of traffic that drops at a location. Units (dependent on network type) are displayed at the top of each column of traffic data. 
Through Traffic RingWorking, MeshWorking, MeshProtectDedicated, Total The amount of each type of traffic that passes through a location.  Units (dependent on network type) are displayed at the top of each column of traffic data. 
Total Traffic   The total amount of traffic routed through a location. Units (dependent on network type) are displayed at the top of each column of traffic data.

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
