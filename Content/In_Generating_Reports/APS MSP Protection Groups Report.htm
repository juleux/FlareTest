APS MSP Protection Groups Report

This report is provided as a worksheet entitled "APS MSP Protection Groups".  It contains port protection information imported from as described in the table below.

Type of Data Column Name(s) Description
Protection Group Name, LongName, Locked, Location This is user supplied information that uniquely describes each protection group.
Admin AdminArchitecture The architecture of a protection group (i.e. OnePlusOne), imported via .
  AdminMode The mode of a protection group (i.e. unidirectional), imported via .
    AdminReversionMode The reversion mode of a protection group (i.e. non-revertive), imported via .
Other OperatingStandard The operating standard (SDH or SONET) of a protection group, imported via .
Working Chassis, Slot, Port The working chassis, slot, and port assigned to a protection group, imported via .
Protect Chassis, Slot, Port The protect chassis, slot, and port assigned to a protection group, imported via .

To generate a report, click on the Export Excel Report icon , or select Tools, Reporting from the menu. Place a checkmark next to the type of report you wish to create, then click OK.
