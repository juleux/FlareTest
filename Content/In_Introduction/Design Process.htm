The Design Process

Using the Modeler, you can design both mesh and ring networks, as well as networks that use a combination of mesh and ring topologies.  The design process for each type of network varies only in the type of design algorithms that are employed.

The process for creating a network design using the Modeler consists of:

  1. Use one of several methods to create a Network file containing your input data.
  2. Route the network using the appropriate algorithm(s).
  3. Use various tools within the Modeler to analyze your design and make changes as needed.
  4. Use the Reporting Tool or the Network Data Tool to generate the desired Excel reports.
