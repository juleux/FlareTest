Locale Converter

Use the Locale Converter to control the distribution of locales when creating a Network View.  To open the Locale Converter, select View, Options from the main menu.  When the Options dialog appears, click to select the Locale Converter tab as shown below.

Locale Converter

Each time you import a network file or create a network view, the Modeler multiplies the Latitude and Longitude values for each location by the corresponding scale (see above).  The resulting values are used as coordinates to place each locale on the map.  Thus, larger scale values result in a wider distribution of locales within a view.  Inverted values have the effect of flipping the locale map either vertically (inverted Latitude), horizontally (inverted Longitude), or both.  To edit a scale value, use the arrow keys or type in a new value then click the Apply button.  To invert a scale value, click to place a checkmark in the Invert sign checkbox.  Each time you make edits in this tab and click OK or Apply, the values you selected will be stored as default values.  If you edit a scale value and decide you want to reinstate the previous value, click on the Defaults button followed by the Apply button.
