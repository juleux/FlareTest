General Viewing Options

Use the General tab of the Options dialog to control both the list of recently accessed files and display appearance.  To access General Viewing Options, select View, Options from the main menu.  When the Options dialog appears, the General tab, shown below, will be selected by default.

General Viewing Options

A description of each option is as follows:

Option Name Description
Last Networks Accessed The Modeler updates a list of the four most recently accessed files and displays it as part of the File menu. To erase the contents of the list, click on the Clear button. 
Display Appearance The Modeler allows you to change the size and position of its various windows.  Click the Defaults button to restore windows to their default size and position.
