Local Demands

A local demand is a demand that starts and ends at a single location.  A local demand is the equivalent of a local circuit as defined by r. 

The following applies to local demands:

  • Local demands are part of the Demands collection.
  • Local demands can be imported from or created within the Modeler.
  • Local demands are displayed in the network view (see example below) as a line that exits a location and loops back to it.  The DemandRouting property of the network view must be set to Direct in order for local demands to be visible.
  • Local Demands are distinguishable from other demands by their property values.  For local demands: 
    • The Local property is set to True. 
    • StartLocation and EndLocation properties have the same value. 
    • Protection and path related properties cannot be edited.
