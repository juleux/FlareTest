Virtual Concatenation Groups

A Virtual Concatenation Group is a logical (rather than physical) combination of one or more circuits within a network that provide paths between two endpoint locations.  To each endpoint location, the circuits assigned to a VCG function as a single Ethernet connection. 

Virtual Concatenation Group

The following applies to Virtual Concatenation Groups in the Modeler:

  • Virtual Concatenation Groups appear in the object tree and are symbolized by this icon: .
  • Ethernet Service objects appear in the object tree as children of Virtual Concatenation Group objects and are symbolized by this icon: .
  • Virtual Concatenation Groups can be created and configured within the Modeler.  They can also be configured within and imported into the Modeler. 
  • When a Virtual Concatenation Group or Ethernet Service object is selected from within the object tree, it's endpoints are highlighted in the network view.
  • A demand can be assigned to any Virtual Concatenation Group whose endpoints and structure are compatible those of the demand.
