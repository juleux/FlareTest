Path Protection Groups

Path Protection Groups (PPGs) provide a method of establishing path level protection for the client ports of a circuit.  Information about a network's Path Protection Groups can be imported into the Modeler for viewing and reporting purposes.

The following applies to path protection groups in the Modeler:

  • Path Protection Groups appear in the object tree as a child object of a local demand and are symbolized by this icon: .
  • PPGs cannot be created or configured within the Modeler.  They are configured within and become part of a network file when network information is imported into the Modeler. 
  • Path Protection Groups are not depicted in network views.
  • The properties of PPG objects are read-only.
