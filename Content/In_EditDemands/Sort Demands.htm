Sort Demands

The Modeler's Sort Demands tool changes the order in which demands are listed in the object tree.  Demands can be sorted by size, name, path property, or randomly.  This tool is useful in two ways:

  • It makes it easier to use the object tree to locate the demands you want to work with.  
  • It creates the potential for a better routing solution by changing the order in which demands are processed by the Modeler's algorithms (algorithms begin processing demands in the order they appear in the object tree). 

To sort demands, click on the Sort Demands icon or select Edit, Demands, Sort... from the menu.  The dialog shown below will appear.  Enter the desired sort parameters then click Sort.
