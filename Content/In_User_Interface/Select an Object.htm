Select an Object

Objects that appear in the Object tree, in the Results window, or in a Network view are available for selection. 

To select an object:

  • The Object Tree - To select an object from the Object tree, click on the object's name or icon.
  • The Results Window - To select an object from a tab within the Results window, click on the object's name.
  • The Network View - To select an object from a Network view, click on the object's name or symbol.  (Note: You may have to drag top level objects aside to access an object that appears on a lower layer of the network view.)

Multiple objects may be selected from within a network view.  You can select multiple objects in one of two ways:

  • Hold down the Shift key (the Cntl key also works) and click on the name or symbol of each object you wish to select.
  • Click the left mouse button and drag a rubberband around the group of objects you wish to select. Any object that is enclosed, in it's entirety, in the selected area, regardless of which layer it appears on, will be included in the selection.

When an object is selected, the following is true:

  • The object is highlighted in the Object tree.
  • The object is highlighted in any Network view in which it is visible.
  • The property values assigned to the object are displayed in the Property editor.  When multiple objects are selected, only the properties that the selected objects have in common will be displayed.
  • The object's type is displayed in the Status bar (labeled Current Object) and in the title bar of the Property editor.

To deselect an object:

  • Click on any object other than the selected object(s).
