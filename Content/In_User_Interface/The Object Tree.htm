The Object Tree 

The Modeler's Object tree contains a hierarchical list of the objects contained in your design.  Use it to select objects throughout the design process.  The contents of the object tree vary depending on the types and properties of the objects present in the network file.  Annotated object tree examples are provided below.

Refer to the Network Model section of this help file for a diagram showing the structure of the object model and a description of each object type. 

When you click on an object in the Object tree, the following is true:

  • The object is selected for editing
  • The object's properties are displayed in the Property editor
  • The object is highlighted in the active Network view 

When you right-click on an object in the Object tree, a list of relevant commands (i.e. Add, Delete, Find Hosted Items, Find Hosts, Network Data Tool), if any, will appear in a pop-up menu (see below). Commands that don't apply to your selection will be grayed out.
