The Main Toolbar 

The Main toolbar at the top of the screen provides easy access to the most commonly used functionality.  It contains the standard Windows icons for managing documents (New, Open, and Save).  Icons unique to the Modeler are labeled below.  To view a tool's description, hover your cursor over it's icon in the toolbar for a few seconds and a tool tip will appear.

Main Toolbar

The Main Toolbar contains icons for a number of tools designed to change the appearance and scale of a network view:

Icon Control Name Description
Zoom In Increases/decreases the Scale property of a view
Zoom Out
Zoom to Original Size Returns a view to its original size by changing the value of its Scale property to one.
Zoom to Fit Changes the scale of a view in order to show all its locales in the current window.
Auto Locales Layout Changes the relative position of locales to provide a best-case view.  Auto layout locales works best when all locales are visible (the Visible property of the Locales object is True) and connected by fiber. 
Align Selected Locales Left Changes the coordinates of a selected set of locales to align them within the view.
Align Selected Locales Right
Align Selected Locales Top
Align Selected Locales Bottom
Spread All Link Items Spreads (curves) overlapping fibers, lightpaths, and demands within a view.  Use this feature to view and select links from multiple layers within a network view. The FibersAsBeziers property of a view must be True for fibers to be spread. The same is true for lightpaths (LightPathsAsBeziers) and demands (DemandsAsBeziers).
Straighten All Link Items Straightens the lines representing fibers, lightpaths, and demands within a view.  
Expand Map Proportionately Changes the relative position of locales by a user specified factor.
Redraw Network Updates the Object tree, Property editor, and the View windows.

 The remaining commands are described in detail in other sections of this help file:

The Object toolbar is located directly below the Main toolbar.  It contains controls for adding components to your network design and is described in the Add Objects section of this document.
