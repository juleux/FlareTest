Find Hosts and Hosted Items

When you right-click on an object in the Object tree, a list of relevant commands (i.e. Add, Delete, Find Hosted Items, Find Hosts), if any, will appear in a pop-up menu (see below).

Use the Find Hosts and Find Hosted Items commands to view a list of objects associated with your selection.  For example, to view a list of the lightpaths and demands that are routed over a specific fiber, right-click on the fiber in the Object tree and select Find Hosted Items from the menu.  The Search Results tab will appear in the Results window.  The fiber you selected will be listed along with the lightpaths and demands it hosts (see below).  When you click on an object in the Search Results list, it becomes the current selection in the Object tree.
