Find Objects

The Modeler's Find Objects tool searches for objects whose Name, LongName, StartLocation, or EndLocation properties contain a string of text that you specify.  To find objects, click on the Find Objects icon or select Edit, Find from the menu.  The dialog shown below will appear.  

Enter your search criteria by selecting one or more object types and fields.  Next, type in the text you wish to search for (Find what:) and click on the Search button.  The Search Results tab will appear in the Results window with a list of the objects that meet your search criteria.
