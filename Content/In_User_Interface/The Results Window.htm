The Results Window

The Results window is located above the Status bar within the Main window.  It contains multiple tabs as shown below.  Each tab contains a list of objects that meet a specific set of criteria (i.e. unrouted demands, failed objects).  To use the Results window, select the desired tab by clicking on it. A list of the objects (if any) that meet the selected criteria will appear.  Click on an object in the list to make it the current selection. Once selected, it will be highlighted in the Object tree and the Network view, and it's properties will be displayed in the Property editor.

A description of each list (tab) is as follows:

  • UnRouted Demands - After a routing algorithm is run, this list contains any demands that the algorithm was unable to route. 
  • NonDiverse Demands - After a routing algorithm is run, this list contains any demands that the algorithm was unable to route in a diverse fashion.
  • Search Results - This list contains the objects that meet your search criteria.  A search can be conducted using the Find Hosts, Find Hosted Items, and Edit, Find commands.
