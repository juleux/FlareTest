Network View Basics

Network views display network data in a graphical, map-like format.  Multiple views can be open at the same time.  When you click on an object in the active network view, the following is true:

  • The object is selected for editing
  • The object's properties are displayed in the Property editor
  • The object is highlighted in the Object tree  

Multiple objects can be selected within a network view.  To select multiple objects, hold down the Shift key and click on each of the desired objects or click and drag a selection box (rubberband) around the desired objects.

To open a network view, click on its name in the Object tree.  To close a network view, click on the close icon located to the right of the menu bar.  Use the Window menu to cascade or tile multiple network views.

Network views are described in detail in the Network Views section of this help document.
