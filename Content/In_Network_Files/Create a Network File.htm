Create a Network File

Network files can be created in the following ways:

  • By entering data in an Excel file using a predefined format and then importing the file into the Modeler. 
  • By importing data from an active network using .   
  • By selecting File, New from the main menu then using Modeler's editing tools to add network objects.  For information on Modeler's editing tools, read the Add Objects section of this help document.
