Import a Network File from Excel

The ability to import data from network files enables you to merge network information from one or more Excel data files.   You can import an entire Excel data file or selected groups of data (worksheets) within a file.  

Modeler imports Excel data into the network file that is open and on display at the time of import.  As a precaution, you should save a backup copy of your network file before attempting to import additional data.  To import network data into the Modeler, select File, Import, from Excel... from the menu.  Select the Excel file you wish to import data from and click the Open button.  The dialog shown below will appear.  

Dialog to Import Data From an Excel File

For each type of worksheet (location, fiber, etc.), the names of the individual worksheets present in the file are listed.  Click the checkbox next to each worksheet that you wish to import or use the Select buttons to select all the worksheets of a given type.  To reverse your selections, click the Toggle All button (upper right).  When your selections are complete, click OK.  Modeler will perform validation checks as it imports your selections.  Any errors that exist will be logged.

Import Options

When the Fix Existing Demand Routing And Set Minimum LightPath Capacities option is checked, demands and lightpaths present prior to import will retain their routing information.  Specifically, each demand's FixedFiberRouting and FixedLightPathRouting properties will be set to True and each lightpath's CapacityMinimum property will be set equal to the value of its Capacity property.  This option is useful when you would like to import data for a later phase of your design.
