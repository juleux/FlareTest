Open/Save an Imported Data Set

The Modeler provides a wizard-like tool to step you through the process of importing network data from a server.   Using this tool, you can connect to a server and view the raw data available for import.  The data, referred to as a data set, consists of a set of tables accessible from the Browse Network Data tab of the Import Network From dialog.

The ability to store and retrieve data sets is helpful in the following situations:

  • You want to capture a snapshot of network conditions at a specific point in time.
  • You want to store network data to work with when your computer is disconnected from your server.
  • You are experiencing error messages during the import process and need assistance in resolving them.  By saving the data set whenever errors are experienced, you can capture current network conditions and share them with others who can assist you in determining the cause of the errors.

Save a DataSet

To save a data set, click on the Save DataSet button in the Browse Network Data dialog.  The Modeler will prompt you to enter the name and location of the data set file (extension .dst) that you wish to create.  When you click on the Save button, the data set file will be stored.

Open a DataSet

To open a previously saved data set:

  1. Select File, Import from the menu.
  2. Click on the Browse Network Data tab.
  3. Click on the Open DataSet button and locate the file you wish to open.  When you select a file and click on Open, the data set will be loaded into the data grid located within the Browse Network Data dialog.
