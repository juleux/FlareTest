Export a Network File to Excel

Network data can be exported to Excel where it can be used in any number of ways.  Network data is exported using the same file format (worksheets, columns) that Modeler accepts when an Excel file is imported.  As a result, it is possible to export, edit, and re-import network data using Excel.  The Create an Excel File for Import section of this document provides a detailed description of the file format Modeler requires for the successful  import of data from Excel.  

To export a file to Excel, select File, Export, Excel... from the main menu.  The dialog shown below will appear.  This dialog enables you to select the type of network information you want to export.  For each type of object, the names of the individual groups present in the file are listed.  Click the checkbox next to each group that you wish to export or use the Select buttons to select all the groups of a given type.  When your selections are complete, click OK.  Each of the groups you selected will be present as worksheets in the destination file.

Note:  Network View objects contain the information Modeler uses to generate a graphical representation of your network.  Network View objects are preserved when you save a network file, but are not available for export to Excel.  The same is true of Failure Scenarios, they can be saved to a network file, but not exported.  Failure Scenarios are used to test the resiliency of a network design.
