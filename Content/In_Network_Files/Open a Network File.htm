Open a Network File

To open a network file in the Modeler, select File, Open from the menu or click on the open file icon .  Select the file you wish to open and click the Open button.  The Modeler performs validation checks on each file as it is opened, such as:

  • Does it contain the required data?
  • Are the name entries unique?
  • Are entries in the proper format, i.e. string, integer, etc.?
  • Do the locations, fibers, and/or lightpaths referenced within the file exist?

The validation process may take a few minutes depending on the size of the file.  If the network file is valid, the Modeler displays a map of the network.  Otherwise, errors are logged as they occur.   

Note:  Two sample network files are provided with the Modeler.  PFL Mesh.xml and PFL Ring.xml are located in the Documents folder contained in the Modeler's install directory.  The default install directory for the Modeler is \Program Files\Sycamore Networks\Modeler.
