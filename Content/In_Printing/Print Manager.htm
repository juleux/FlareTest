Print Manager

Use the Print Manager to print one or more network views (they need not be open).  To use the print manager, select File, Print Manager from the main menu. The dialog shown below will appear.  Click to select the network views you want to print then click Print.

Print Manager Dialog

Print NetworkViews When checked, network views are selected for printing.
All Click to select all of the available network views for printing.
None Click to deselect all of the available network views for printing.
   
Preview only Check to view a Preview of your selections.
Page Setup Click to edit Page Setup options.
Print Click to print the selected network views.
Close Click to close the dialog.
