Print Network Views

The Modeler provides tools for previewing and printing the network views you have created to represent your network design.  To print a network view that is open, click within the view window and select File, Print from the main menu.  To print one or more network views (they need not be open), select File, Print Manager from the main menu. 

To print network information in tabular form (as opposed to a network view), use the Network Data Tool or the Reporting Tool to create spreadsheet files that may be formatted and printed using Excel.
