Page Setup

Use the Page Setup dialog to do the following:

  • Select paper size - select a size from a list of available sizes
  • Select printer paper source - select a source from a list of available trays
  • Select paper orientation - click on landscape or portrait
  • Select page margins - enter margin size in inches for left, right, top, and bottom
  • Select a printer - click on the printer button and select from a list of available printers

To change page setup options, select File, Page Setup from the menu.
