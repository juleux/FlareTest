Manual Time Slot Editing - Mesh

This help section describes the Manual Time Slot Editing tool and the process for manually editing the time slot assignments for a selected demand.  It contains instructions for opening the editor and using it to edit existing time slot assignments.

Open the Manual Time Slot Editor

The Modeler's Manual Time Slot Editor provides tools to view and edit the time slots assigned to individual demands.  To use the editor:

  1. Click on the name of the demand you wish to edit in the Object tree.
  2. Locate the WorkingPathStartingTimeSlots property (or the ProtectPathStartingTimeSlots property, if the demand type is OnePlusOne) and click on it.  A button with … in it will appear on the right edge of the box.
  3. Click on the ellipsis … button (as shown below).
  4. The manual time slot editor will appear.  The editor will display the time slot information for the demand you selected.

Edit a Time Slot Assignment

To edit a time slot assignment, you must first select a demand and open the time slot editor.  A screen capture of the Manual Time Slot Editor is shown below.  The editor contains a list of the lightpaths that comprise the selected demand's path along with the starting time slot.  To edit a time slot assignment:

  1. Click on the Demand Quantity pull-down to select the demand index (for those demands of quantity > 1) you wish to edit.
  2. Click on the StartingTimeSlot pull-down to select from a list of the starting time slots available for assignment.  If the desired time slot is not listed, it is being used by another demand.  To view a list of the time slots assigned to a particular lightpath, use the Network Data Tool.
  3. Choose a value for the Fix LightPath TimeSlots option. When the Fix LightPath TimeSlots option is checked, both the FixedLightPathTimeSlots property and the FixedLightPathRouting property of the demand being edited will be set to True when you exit the time slot routing tool.  By using this default option, you prevent the time slots assigned during the editing process from being overwritten the next time a routing algorithm executes. 
  4. When your edits are complete, click on the OK button to close the editor.
