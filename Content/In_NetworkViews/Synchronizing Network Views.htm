Synchronizing Network Views

Synchronization changes the layout (locale coordinates, visibility, and icon properties) of a network view to match the layout of a base view of your choice.  It can also change a network's location coordinates (latitude and longitude properties) to match the locale coordinates (X and Y properties) of a selected (base) view.  Examples of situations where synchronization might be helpful include:

  • You want multiple network views to have the same locale properties.
  • You want all new network views to have a specific layout.

When a new view is created, Modeler calculates each locale's coordinates using the Latitude and Longitude properties of its corresponding location along with the offset values defined in the Locale Converter.  As a locale is repositioned within a view, its X and Y properties are updated accordingly, but the coordinates of its corresponding location remain unchanged.  When you synchronize with the Location Latitude and Longitude option selected, each location's coordinates are overwritten with the X and Y properties of its corresponding locale within the base view.  Any new views that are then created (with locale latitude and longitude conversion values set equal to one) will have the same locale positioning as the base view.

To synchronize one or more network views, select View, Synch NetworkViews from the main menu. The dialog shown below will appear.  Enter the information as described below then click OK.
