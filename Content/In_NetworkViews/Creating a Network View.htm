Creating a Network View

To create a network view, click on the Network View icon in the Object toolbar, or select Edit, Add, View from the menu.  Provide a name for your view and click OK.  The result is as follows:

  • A network view object is added to the object tree as a member of the NetworkViews collection. 
  • A locale is created for each location in the network. Each locale has the same name as its corresponding location. 
  • The initial coordinates for each locale are derived from the latitude and longitude values of its corresponding location and user specified offsets (Locale Converter).  
  • The newly created view is displayed. 

To change the relative placement of locales within a new view, adjust the latitude and longitude values in the Locale Converter tab of the View, Options dialog.  Making changes to these values has no affect on the locale coordinates of existing views.
