Network Views

A network view is a graphical representation (bird's eye view) of a network design.  The Modeler provides network views to help you manage the large amounts of data needed to design a network.  By creating multiple views each configured to display different types of network data, you can filter out the information that is important to you and speed up the design process.  When a new project is created in Modeler, it contains a default view for each layer of information; fiber, lightpath, and demand.

The following applies to network views:

  • Each network view contains a map of locales (one or more nodes) as a base layer.  You can display additional layers showing the fibers, lightpaths and/or demands that interconnect your network's locales. 
  • You can create multiple views of the same network.
  • You can pan and zoom within a network view. 
  • You can customize the appearance of a network view by editing it's properties, layout, or scale. 
  • You can synchronize one or more network views to a selected (base) view.  Synchronization changes a network view's layout (locale coordinates, visibility, and icon properties) to match that of the base view.
  • You can print one or more network views.
